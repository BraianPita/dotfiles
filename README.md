# dotfiles
My dotfiles


## Setup

### Linux Steps

Install zsh 
```
sudo apt install zsh
```

Install ohmyposh
```
curl -s https://ohmyposh.dev/install.sh | bash -s
```

Install zoxide
```
curl -sSfL https://raw.githubusercontent.com/ajeetdsouza/zoxide/main/install.sh | sh
```

Install fzf
```
git clone --depth 1 https://github.com/junegunn/fzf.git ~/.fzf
~/.fzf/install
```
